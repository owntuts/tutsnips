## What Is C# (C-Sharp)

C# (pronounced as “See Sharp”) is a modern, object-oriented, type-shape programming language. C# allows developers to build a wide variety of secure and powerful applications that run in .NET. C# is derived from the C family of languages ​​and will be immediately familiar to C, C, Java and JavaScript programmers.

## C# Features

<Img src="/img/tutorial/c-sharp-tut/c-sharp-features.jpg" />     

<div className='list-askt pdl'>

- Garbage collection automatically reclaims memory occupied by unreachable unused objects.
- Nullable types guard against variables that don’t refer to allocated objects.
- Exception handling provides a structured and extensible approach to error detection and recovery.
- Lambda expressions support functional programming techniques.
- Language Integrated Query (LINQ) syntax creates a common pattern for working with data from any source.
- Language support for asynchronous operations provides syntax for building distributed systems.

</div>

## C# Pros & Cons

### Pros

<div className='list-thumb-up pdl'>

- C-family of languages
- Object-oriented
- Inbuilt garbage collector
- Type-safe language with dynamic capabilities
- Thorough documentation
- Visual Studio as a native IDE
- Large community

</div>

### Cons

<div className='list-thumb-down pdl'>

- C# performance isn’t the absolute best
- Dependence on .NET platform
- Hard learning curve

</div>
